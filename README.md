## 🚀 Quick Start

### Backend (Port 8080)
```bash
cd backend
npm install
npm run dev
```

### Frontend (Port 5173)
```bash
cd frontend
npm install
npm run dev
```

**For detailed setup instructions, see [docs/SETUP.md](docs/SETUP.md)**
